    ABS Data Fixtures

    Data in this directory was sourced from the Australian Bureau of Statistics (ABS) via the ABS Data API (https://api.data.abs.gov.au).

    Attribution: Australian Bureau of Statistics, accessed April 2026. Licensed under Creative Commons Attribution 4.0 International (CC BY 4.0). https://www.abs.gov.au/website-privacy-copyright-and-disclaimer

    Datasets

    File Source Dataflow
    gdp_readings.json National Accounts ABS,ANA_AGG
    labour_force_readings.json Labour Force, Australia ABS,LF
    cpi_readings.json Consumer Price Index ABS,CPI
    leading_index_readings.json Business Indicators (proxy) ABS,BI
    daily_activity_readings.json Generated (no ABS equivalent) —

    Notes

    • GDP and Labour Force data cover 2000–2024
    • CPI covers 2000–2024 by expenditure category
    • The leading index uses ABS Business Indicators as a proxy — the Westpac-Melbourne Institute Leading Index is not published via the ABS API
    • Daily activity data remains procedurally generated as no equivalent ABS daily series exists

    Data structure notes

    The ABS API returns SDMX-formatted CSV. The fetch script transforms this into flat JSON matching our database schema. Industry and category names are taken directly from ABS codelist labels.
